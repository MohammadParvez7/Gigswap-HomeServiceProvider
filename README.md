# GigSwap – A Home Service Provider
- Built a web-based Home Service Provider platform using the MERN Stack.
- Provided multiple local services such as Electrician, Painter, and more.
- Delivered a clean and responsive user interface for a smooth experience.
- Technologies:- React.js , Node.js , Express.js , Mongodb.
